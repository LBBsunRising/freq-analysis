# 频域局部对比分析工具

> 本项目由 **GLM-5.2 纯 vibe coding** 生成 —— 代码全部由 GLM-5.2 在 Kilo 中通过自然语言对话驱动编写，未经过人工手写或修改。

一套用于在图像上框选任意区域、计算其 FFT 幅度谱并与背景对比的工具，辅助判断目标纹理在频域上是否可分。包含交互式 GUI 与命令行基础版两个版本。

## 功能概览

### frequency_gui.py —— 频谱分析 GUI（核心）

基于 PySide6 的交互式工具：

- **多图浏览**：工具栏「打开图片」多选，或「打开文件夹」导入整个目录
  - 打开文件夹时弹出选择窗口：列出所有图片文件、可勾选 / 筛选 / 预览缩略图 / 限制加载数量，避免一次导入过多导致崩溃
- **拖拽框选**：在左侧画布上鼠标拖拽框选任意区域，松开即生成该区域的频谱图
  - 支持连续框选、任意数量
  - 滚轮缩放，左 / 右方向键或工具栏切换图片
- **全局结果面板**：所有图片的频谱结果集中显示在右侧，**不随图片切换清空**，按框选顺序排列，并标注来源图片名
- **图像增强**：底部面板可调亮度 / 对比度 / 伽马 / 锐化 / CLAHE / 反色
  - **所见即所分析**：调整同时作用于显示与频谱计算
  - 每张图独立增强参数，可「应用到全部」
- **导出**：将所有频谱图、空间 patch 导出为 PNG，框选坐标 + 增强参数汇总为 CSV
- **内存保护**：图片懒加载，切换时自动释放非当前图原图，可稳定导入大量图片

### frequency-analsys.py —— 频谱分析基础版

基于 cv2 + matplotlib 的命令行版本：交互式框选目标区域与背景区域，并排对比两者的空间 patch 与频谱图。

## 使用方法

启动 GUI：

```bash
python frequency_gui.py
```

运行基础版（需自行修改脚本中的图片路径）：

```bash
python frequency-analsys.py
```

## 环境依赖

- Python 3.10+
- OpenCV (`opencv-python`)
- NumPy
- PySide6（GUI）
- Matplotlib（基础版）

安装：

```bash
pip install opencv-python numpy PySide6 matplotlib
```

## 频谱可视化原理

对框选区域转灰度后做二维 FFT，频谱中心化（`fftshift`），取对数幅度谱（`20·log(|F|+1)`）并归一化到 0–255，再用 `COLORMAP_INFERNO` 伪彩色映射，高频成分以暖色凸显，便于肉眼对比目标与背景的纹理频率分布。
